# 📰 El Faro - Sitio Web de Noticias

Proyecto desarrollado para la **Actividad Sumativa Semana 5 - AIEP**.

---

## 🚀 Descripción

Sitio web de noticias "El Faro", rediseñado con una estética moderna utilizando el framework **Bulma**, incorporando organización por secciones, contenido dinámico y elementos multimedia.

---

## 🛠 Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Bulma (Framework CSS)

---

## ✨ Funcionalidades

* ✔ Diseño moderno y responsivo
* ✔ Menú de navegación por secciones
* ✔ Sección de noticias destacadas
* ✔ Sistema dinámico para agregar artículos
* ✔ Contador automático de artículos
* ✔ Fecha y hora en tiempo real
* ✔ Validación de formulario de contacto
* ✔ Integración de video y audio

---
